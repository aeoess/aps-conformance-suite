# envoys-rfc9421 composition fixture

Three deterministic vectors demonstrating per-message RFC 9421 wire signing composed with APS receipt and delegation layers. L1 is a per-message RFC 9421 HTTP Message Signature over jschoemaker's @envoys/sdk v1.4.0 §13 Vector 2 (`POST /api/task`) signed with the RFC 8032 §7.1 Test 1 Ed25519 keypair, and byte-matches the spec's expected `Signature` header. L2 wraps the L1 signature inside an APS `bilateral_receipt` envelope (`~/agent-passport-system/src/types/bilateral-receipt.ts`); both the requesting agent (RFC 8032 §7.1 keypair, same as L1) and the serving agent (seed = 32 bytes of `0x01`) counter-sign the same canonical body, and the L1 signature value is bound into `evidenceCommitments[0].credentialHash`. L3 wraps L2 inside a 3-link APS delegation chain (`~/agent-passport-system/src/v2/types.ts` `V2Delegation`) with seeds `0x02` / `0x03` / `0x04` for root / middle / leaf signers; `action_categories` narrow monotonically (`{checkout, refund, dispute}` → `{checkout, refund}` → `{checkout}`), the leaf's `scope.constraints.bilateral_receipt_ref` binds to L2's `receiptId`, and the leaf's delegatee is the wire-signing agent that produced L1.

Run `npm run verify:envoys-rfc9421` from the repo root. The verifier re-derives every signature from the seeds and fixed inputs in `lib.ts`, compares byte-for-byte against the committed `L1.fixture.json` / `L2.fixture.json` / `L3.fixture.json`, runs a live Ed25519 verify on every signature against the documented public key, checks monotonic narrowing across the chain, and confirms the L1 → L2 → L3 binding hashes. Exit code 0 on full byte-match across all three layers, 1 on any mismatch with a diff line. `generate.ts` is the one-shot script that produced the fixtures; re-running it yields identical bytes given the lib's seeds and timestamps. Pinned to `@envoys/sdk@0.7.0` (the v1.4.0 reference SDK with `Envoys.resolveDidWeb()` shipped); spec at `https://github.com/jschoemaker/Envoys-public/blob/main/specs/signature/v1.md`; A2A discussion at A2A#1829.
